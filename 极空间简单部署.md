# 极空间NAS简单部署（无需Docker）

最简单的方法：直接将文件放到NAS，通过极空间的文件服务访问。

## 🚀 超简单部署步骤

### 步骤1：上传文件到NAS

1. **打开极空间客户端**（手机App或电脑客户端）
2. **登录您的NAS账号**
3. **进入"文件管理"**
4. **找到或创建Web目录**：
   - 通常极空间有一个 `web` 或 `www` 文件夹
   - 如果没有，可以创建一个 `web` 文件夹
   - 或者直接放在任何共享文件夹中

5. **上传整个study文件夹**
   - 将整个 `study` 文件夹复制到NAS的 `web` 目录
   - 或者上传到您选择的共享文件夹

### 步骤2：通过极空间访问

#### 方法一：通过极空间客户端访问

1. **打开极空间客户端**
2. **进入文件管理**
3. **找到 `study` 文件夹**
4. **点击 `index.html` 文件**
5. **选择"在浏览器中打开"**

#### 方法二：通过极空间Web界面访问

1. **在浏览器中访问极空间Web界面**
   - 通常是：`http://极空间IP地址` 或 `https://极空间IP地址`
   - 或使用极空间提供的域名

2. **登录极空间管理界面**

3. **进入"文件"或"文件管理"**

4. **找到 `study` 文件夹**

5. **点击 `index.html`**
   - 文件会在浏览器中打开
   - 或右键选择"在新标签页中打开"

### 步骤3：访问游戏

- **游戏列表**：打开 `study/index.html`
- **舒尔特方格**：打开 `study/schulte-grid/index.html`

---

## 📱 在iPad上使用

### 方法一：通过极空间App

1. **在iPad上安装极空间App**
2. **登录您的NAS账号**
3. **进入文件管理**
4. **找到 `study/index.html`**
5. **点击文件，选择"在Safari中打开"**
6. **在Safari中添加到主屏幕**

### 方法二：通过Safari直接访问

1. **在iPad Safari中访问极空间Web界面**
2. **登录后进入文件管理**
3. **打开 `study/index.html`**
4. **添加到主屏幕**

---

## ⚙️ 配置极空间Web服务（可选）

如果想让应用通过固定网址访问：

### 启用极空间Web服务

1. **登录极空间管理界面**
2. **进入"应用中心"或"服务"**
3. **查找"Web服务"或"文件服务"**
4. **启用Web服务功能**
5. **设置Web根目录**（通常是 `/web` 或 `/www`）

### 访问方式

如果启用了Web服务，可以通过：
- `http://极空间IP/web/study/index.html`
- 或 `http://极空间IP/study/index.html`（如果study在根目录）

## 🔒 关于HTTPS和PWA

### 重要提示

**如果只是通过文件服务访问：**
- ✅ 基本功能可以使用
- ✅ 训练功能正常
- ✅ 成绩记录正常
- ❌ **PWA功能可能受限**（需要HTTPS）
- ❌ **无法添加到主屏幕**（需要HTTPS）
- ❌ **离线功能可能无法使用**（需要HTTPS）

### 如果需要完整PWA功能

如果需要在iPad上安装并使用离线功能，建议：
1. **使用Docker部署**（参考 `部署到极空间NAS.md`）
2. **配置HTTPS**（PWA必需）
3. **或使用GitHub Pages等免费托管**（支持HTTPS）

## 📂 推荐的文件结构

在NAS上建议这样组织：

```
极空间共享文件夹/
└── web/                    # Web服务目录
    └── study/              # 学习训练游戏
        ├── index.html      # 游戏列表
        ├── schulte-grid/   # 舒尔特方格
        └── [其他游戏]/     # 后续添加的游戏
```

## 📝 注意事项

1. **文件路径**：确保所有文件都在同一目录下
2. **相对路径**：游戏内的文件使用相对路径，确保结构正确
3. **权限**：确保文件有读取权限
4. **缓存**：如果更新了文件，清除浏览器缓存
