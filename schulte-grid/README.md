# 舒尔特方格训练软件

一个专为小学生设计的注意力训练工具，帮助提升专注力和减少粗心错误。

## 功能特点

- 🎯 **多种难度选择**：从3×3到7×7，适合不同水平的学生
- ⏱️ **精确计时**：毫秒级计时，记录每次训练成绩
- 🏆 **成绩记录**：自动保存最佳成绩和历史记录
- 📊 **训练历史**：查看过往训练记录，追踪进步
- 🖨️ **打印功能**：支持打印练习纸，离线训练
- 🎨 **友好界面**：色彩丰富，适合小学生使用
- 📱 **响应式设计**：支持电脑、平板、手机访问
- 📲 **PWA支持**：可安装到iPad，像原生应用一样使用
- 🔌 **离线使用**：安装后完全离线可用

## 使用方法

### 在线使用

1. 进入游戏目录：
   ```bash
   cd schulte-grid
   ```

2. 直接用浏览器打开 `index.html` 文件
   - 或者使用本地服务器：
   ```bash
   # Python 3
   python3 -m http.server 8000
   # 然后访问 http://localhost:8000
   ```

3. 选择难度（建议三年级从5×5开始）
4. 点击"开始训练"按钮
5. 按照数字顺序（1, 2, 3...）快速点击方格中的数字
6. 完成后查看成绩和评价

### 📱 安装到iPad（PWA版本）

本应用支持作为PWA安装到iPad，体验类似原生应用！

**快速安装步骤：**

1. **生成图标**（首次使用）：
   ```bash
   pip3 install Pillow
   python3 generate_icons.py
   ```

2. **安装到iPad**：
   - 在iPad Safari中打开应用
   - 点击分享按钮 → "添加到主屏幕"
   - 完成！

详细说明请查看 [README_IPAD.md](README_IPAD.md) 或 [IPAD_INSTALL.md](IPAD_INSTALL.md)

**PWA功能：**
- ✅ 离线使用（首次访问后）
- ✅ 全屏体验（无浏览器地址栏）
- ✅ 快速启动（从主屏幕）
- ✅ 自动更新

## 训练建议

### 适合年龄和难度

- **一年级**：建议从3×3开始
- **二年级**：建议4×4或5×5
- **三年级**：建议5×5，目标30-50秒
- **四年级及以上**：可以挑战6×6或7×7

### 训练频率

- 每天1-2次
- 每次3-5分钟
- 坚持练习效果更佳

### 训练环境

- 保持安静的环境
- 减少外界干扰
- 确保光线充足
- 使用电脑或平板效果更好

## 成绩评价标准

### 5×5方格（三年级推荐）

- ⭐ **优秀**：30秒以内
- 👍 **良好**：30-50秒
- 💪 **合格**：50-70秒
- 🎯 **需努力**：70秒以上

## 技术说明

- 纯前端实现，无需后端服务器
- 使用LocalStorage保存数据
- 支持现代浏览器（Chrome、Firefox、Safari、Edge）
- 响应式设计，适配各种屏幕尺寸

## 文件说明

### 核心文件
- `index.html` - 主页面（支持PWA）
- `style.css` - 样式文件（优化iPad触摸）
- `script.js` - 核心功能逻辑（包含Service Worker）

## 注意事项

- 数据保存在浏览器本地，清除浏览器数据会丢失记录
- 建议定期备份重要数据
- 打印功能需要浏览器支持打印
